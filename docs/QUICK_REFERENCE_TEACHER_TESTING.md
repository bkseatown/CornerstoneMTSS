## RED FLAGS DURING TESTING

**STOP & ESCALATE IMMEDIATELY**
- Multiple teachers reporting same bug
- Data loss or corruption
- Teachers unable to log in
- Game crashes
- Platform completely down (>30 min)

**PLAN FIX TODAY**
- Recommendation engine returning bad advice
- Evidence not saving
- Critical feature broken
- Mobile experience broken
- Multiple teachers reporting same issue

**LOG & MONITOR**
- Single teacher has minor bug
- Slow performance
- Accessibility issue on specific browser
- Feature request
- Question about expected behavior

## COMMON ISSUES & FIXES

### "I don't see demo students"
→ Clear browser cache (Cmd+Shift+Delete) + refresh

### "Game won't start / blank screen"
→ Refresh page (Cmd+R), try different game, clear service worker

### "Slow loading"
→ Normal on slow WiFi. Try mobile hotspot. Report if persistent.

### "Dark mode text hard to read"
→ Known issue on some browsers. Switch to light mode temporarily.

### "I can't log in"
→ Check credentials. Clear cache. Try different browser. Email immediately.
