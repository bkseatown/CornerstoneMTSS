## Archived Collaboration Slice

This folder contains the staged collaboration implementation that was extracted from the active repo surface on 2026-03-19.

Why it was archived:

- no shipped HTML page imported the collaboration frontend files
- the browser client and server were not protocol-compatible
- the active product and test flows do not depend on this slice

What is here:

- frontend collaboration files from `js/collab/`
- overlay UI from `games/ui/collab-overlay.js`
- collaboration styles from `style/collab-overlay.css`
- the standalone collaboration backend from `server/`
- collaboration-specific guides moved out of `docs/`

Status:

- archived for reference or future extraction
- not part of the active frontend runtime path
